# Theme Toggler

A simple website with theme-toggler that allows users to toggle between light and dark themes on the website.

## Features

- Toggle between light and dark themes
- Smooth transition between themes
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- React
- Vite
- Tailwind CSS

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/12Sushree/theme-toggler.git
   ```
2. Navigate to the project directory:
   ```bash
   cd theme-toggler
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:5173` to see the website.

## Usage

- Click on the toggle button to switch between light and dark themes.
